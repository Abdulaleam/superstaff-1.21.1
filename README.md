# Core Items

**Magic Gem**

![image](https://cdn.hackclub.com/019e6e63-7d17-7484-ab19-d652da1f7ed9/image.png)

---

**Arcane Runes (Magic Station)**


![image](https://cdn.hackclub.com/019e6e63-d1de-7f07-acd6-278ad8dda780/image.png)
